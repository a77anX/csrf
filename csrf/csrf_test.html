CSRF Attack Test Page

How to check the CSRF token:

  1. First, login here (open in new tab)
  2. After logging in, come back to this page
  3. Press F12 to open Developer Tools
  4. Go to the Network tab
  5. Click the "Try CSRF Attack" button below
  6. In the Network tab, look for the request to "process_profile_update.php"
  7. Click on the request to see its details
  8. Look under the "Form Data" or "Request Payload" section

Note: You won't see the CSRF token in this request because this is the attack page. To see the actual CSRF token, go to the real update profile page after logging in.
